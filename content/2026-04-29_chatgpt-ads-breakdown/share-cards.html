🚨 到底发生了什么？

OpenAI 已经在 ChatGPT 中上线了广告系统。


当你在 ChatGPT 聊天时，后台会根据你的对话内容，在模型回复的数据流中直接插入广告卡片。


这意味着：


① 你的对话内容被用来做广告定向

② 广告像模型回复一样自然地出现

③ 点击后被加密 token 追踪 30 天

④ 已有品牌入驻：Grubhub、Canva、GetYourGuide…

💡 这不是传言——是安全研究人员基于真实网络流量抓包的完整技术分析。

⚙️ 技术实现：SSE 流注入

广告通过 ChatGPT 的 SSE 流（Server-Sent Events）直接注入到模型回复中：

event: delta
data: {
  "type": "single_advertiser_ad_unit",
  "advertiser_brand": {
    "name": "Grubhub",
    "id": "adacct_6984ed..."
  },
  "carousel_cards": [{
    "title": "Get Chinese Food",
    "target": { "url": "..." }
  }]
}

关键细节：


• 广告是独立数据类型，不是模型生成的文本

• 素材托管在 bzrcdn.openai.com

• 默认在 ChatGPT 内置浏览器中打开

🔐 4 层加密追踪

每个广告携带 4 个 Fernet 加密 token：


❶ ads_spam_integrity_payload

→ 防伪造点击校验（不在URL中）


❷ oppref

→ 核心归因 token → 写入 cookie 30 天


❸ olref

→ 出站链接日志（仅URL中）


❹ ad_data_token

→ 嵌套加密，服务端数据核对

闭环：你点击 → OAIQ SDK 种 cookie → 你的后续浏览/购买行为 → 上报回 OpenAI → 归因完成 ✓
4/6 @Guang Yang
05 / 06

🛡️ 怎么屏蔽？

方法一：拦截广告域名

bzrcdn.openai.com
bzr.openai.com

添加到 uBlock Origin 或系统 hosts 文件


方法二：清除追踪 Cookie

__oppref（30天归因追踪）
__oaiq_domain_probe（域名探测）

方法三：用 API 替代网页版

API 调用暂未发现广告注入，是最干净的方式。


方法四：注意 open_externally

如需点击广告链接，复制到独立浏览器打开。

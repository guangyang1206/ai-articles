## 修正清单

### FIX-01 | 类型：事实错误 | 置信度：高

**原文：**
```
亚马逊2026年AI基础设施投资计划高达2000亿美元
```

**改为：**
```
亚马逊2026年资本支出约2000亿美元，其中绝大部分投向AI基础设施
```

**来源：** Amazon CEO Jassy 致股东信原文表述为"total capex ~$200B"，非仅AI基础设施。全年总capex包含物流、AWS通用基础设施等。

### FIX-03 | 类型：数据不精确 | 置信度：中

**原文：**
```
一个10万张H100级别的AI集群，单是GPU采购就可能接近数十亿美元；若算上服务器整机、高速网络、存储、电力、冷却、土地和运维，完整数据中心往往是百亿美元级工程。
```

**改为：**
```
一个10万张H100级别的AI集群，单是GPU采购就约300—400亿美元（按H100单卡3—4万美元计）；若算上服务器整机、高速网络、存储、电力、冷却、土地和运维，完整数据中心总成本可达400—500亿美元级别。
```

**来源：** H100 单卡约 $3-4 万（行业共识），10万张 = $300-400亿 GPU alone。"数十亿"严重低估。

## 仍需修正项

### FIX-07 | 类型：数据口径不精确 | 置信度：高

**原文：**
```
亚马逊2026年AI基础设施投资计划高达2000亿美元
```

**改为：**
```
亚马逊2026年资本支出约2000亿美元，绝大部分投向AI基础设施
```

**原因：** 据亚马逊CEO贾西致股东信及多家机构测算，$2000亿是全年总capex，其中约60-70%（约$1200-1400亿）用于AWS AI基础设施，15-20%用于自研芯片，其余用于卫星和物流自动化。文章将其表述为"AI基础设施投资计划"不完全准确，会误导读者以为$2000亿全部用于AI基础设施。

数据卡同样需改：

**原文：**
```
$2000亿 | 亚马逊AI计划
```

**改为：**
```
$2000亿 | 亚马逊2026年capex
```

**来源：**
- 环球网/路透社：贾西致股东信"重申2026年约2000亿美元资本支出计划，绝大部分投向AI基础设施"
- Longport/花旗：2000亿中约75%（1500亿）用于AWS基础设施
- lc-ceo.cn："2000亿美元的AI资本支出将成为亚马逊2026财年总资本支出的核心组成部分"

### FIX-09 | 类型：事实状态过时 | 置信度：高

**原文：**
```
西雅图市议会正在推进一项为期365天的数据中心暂停审批令
```

**改为：**
```
西雅图市议会已通过一项为期365天的数据中心暂停审批令
```

**原因：** 该禁令4月30日提出，5月委员会通过，6月3日全会表决。文章发布于6月7日，此时禁令已表决通过，不应再说"正在推进"。如不确定表决结果，可改为"西雅图市议会推进了一项为期365天的数据中心暂停审批令"。

**来源：** Seattle City Council legislative calendar

## 仍需修正项

### FIX-10 | 类型：表述不精确 | 置信度：中

**原文：**
```
— 2021年：种子轮，1.24亿美元
```

**改为：**
```
— 2021年5月：A轮，1.24亿美元
```

**原因：** $124M是Series A（A轮），非种子轮。种子轮通常指公司成立前的极早期融资，Anthropic的A轮已有$10亿级投前估值，属于标准A轮。

### FIX-11 | 类型：🔴严重事实错误 | 置信度：高

**原文：**
```
— 2024年：G轮，估值约600亿美元
```

**改为：**
```
— 2024年3月：D轮，27.5亿美元，估值184亿美元
— 2024年11月：Amazon追加投资40亿美元
```

**原因：** 2024年没有G轮。2024年的实际融资是：
- 3月：Series D，$2.75B，估值$18.4B（Amazon完成$4B承诺）
- 11月：Amazon追加$4B（累计$8B）

G轮（Series G）发生在**2026年2月**，$300亿，估值$3800亿。

**来源：** TechMarketBriefs、36kr、新浪财经、Morningstar均确认此时间线。

### FIX-12 | 类型：🔴严重事实错误 | 置信度：高

**原文：**
```
— 2025年：H轮，650亿美元，投后估值9650亿美元
```

**改为：**
```
— 2025年3月：E轮，35亿美元，估值615亿美元
— 2025年9月：F轮，130亿美元，估值1830亿美元
— 2026年5月28日：H轮，650亿美元，投后估值9650亿美元
```

**原因：** H轮完成于2026年5月28日，不是2025年。2025年发生了两轮重要融资（E轮和F轮），文章完全遗漏。遗漏E轮和F轮导致读者无法理解估值从$18.4B→$965B的跃升路径。

**来源：** 新浪财经6月2日报道："5月28日，Anthropic刚宣布完成650亿美元H轮融资"；极客公园："就在前两天的5月28日，Anthropic刚宣布完成650亿美元H轮融资"。

### FIX-13 | 类型：事实不精确 | 置信度：高

**原文：**
```
2026年6月1日，Anthropic向SEC提交了S-1注册文件，正式启动IPO流程
```

**改为：**
```
2026年6月1日，Anthropic向SEC秘密提交了S-1注册声明草案，正式启动IPO筹备流程
```

**原因：** 提交的是保密S-1草案（confidential draft），不是正式S-1注册文件。保密草案允许公司在公开财务数据前与SEC进行内部审核。正式招股书需待SEC审核完成后才能公开。这一区别对理解IPO进度很重要。

**来源：** 新浪财经6月2日："Anthropic官宣已秘密递交招股书！官方公告称，已于6月1日向SEC提交了S-1表格的初步注册声明"；极客公园："Anthropic保密向SEC提交了S-1注册声明草案"。
